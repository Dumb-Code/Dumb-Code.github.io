Project Nublar


About

Project Nublar is a mod that adds Dinosaurs to the world of Minecraft. It aims to bring creatures canon to the Jurassic Park novel and movie franchise in the game.



Requirements

Project Nublar Requires DumbLibary, which can be found on it's mod page here



Release Date

DumbCode plans to release a single beta for Project Nublar for Forge 1.12.2, after which all development will move to the lastest forge version.

No dates have been set for release or the beta.
