Project Nublar


About

Project Nublar is a mod that adds Dinosaurs to the world of Minecraft. It aims to bring creatures cannon to the Jurassic Park novel and moview franchise to life in the game.



Requirements

Project Nublar Requires DumbLibary, which can be found on it's mod page here



Release Date

DumbCode plans to release a beta for Project Nublar for Forge 1.12.2

A full release of Project Nublar is going to be scheduled for a new version of minecraft, the team is undecided between 1.14.4-1.16.1 but we will keep the community up to date on information as we decide on it

No dates have been set for release or the beta.
